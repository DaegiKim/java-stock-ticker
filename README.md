# java-stock-ticker

![java-stock-ticker](https://raw.githubusercontent.com/DaegiKim/java-stock-ticker/master/screenshot.gif)

## Requirements
- Java 10+
- Dependencies
  - org.apache.httpcomponents:httpclient:4.5.9
  - org.json:json:20180813
